# TOPIK II 53번 쓰기 실전 연습

Professional web application for TOPIK II Question 53 writing practice.

## Features

✅ **Header Section**
- Title display
- Name input field
- 50-minute countdown timer
- Dark mode toggle

✅ **Question Area**
- Display 53-style opinion topic
- Random topic selection button
- 6 predefined topics

✅ **Image Upload**
- Drag & drop functionality
- Click to upload
- Image preview with resize slider
- Remove image button
- Supports PNG, JPG, JPEG

✅ **Writing Grid (원고지)**
- 35 rows × 20 columns = 700 boxes
- 1 character per box
- Live character counter
- Warning below 600 characters
- First box empty rule
- Keyboard navigation (arrows)

✅ **Writing Analytics**
- Writing time tracking
- Character count
- Characters per minute

✅ **Export Features**
- PDF export with full layout
- PNG image export
- Includes name, topic, image, grid, and date

✅ **UI Design**
- Modern Korean exam style
- Rounded corners and soft shadows
- Responsive layout
- Dark mode support

## Files

- `index.html` - Main HTML structure
- `style.css` - Complete styling with dark mode
- `script.js` - All functionality and logic

## Usage

1. Open `index.html` in a web browser
2. Enter your name
3. Click "새 주제 받기" to get a random topic (optional)
4. Upload an image if needed
5. Click on the grid and start typing
6. Use arrow keys to navigate
7. Export to PDF or PNG when done

## Technologies

- Pure HTML5
- CSS3 with CSS Variables
- Vanilla JavaScript (ES6+)
- html2canvas for screenshots
- jsPDF for PDF generation
